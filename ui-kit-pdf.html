01. Fundamentos

Principios de diseño que guían la creación de experiencias elegantes y profesionales para el coaching

Filosofía de Diseño

Elegancia Premium: Colores sofisticados y tipografías refinadas que transmiten profesionalismo y confianza.

Facilidad de Uso: Interfaces intuitivas que permiten a los clientes enfocarse en su transformación personal.

Consistencia Visual: Experiencia unificada en todas las páginas y dispositivos.

Arquitectura del Sistema

• Tailwind CSS v4: Framework utilitario para desarrollo ágil
• CSS Variables: Personalización fácil de colores y espaciados
• Mobile First: Diseño responsive optimizado para todos los dispositivos
• Performance: CSS optimizado y fuentes con carga inteligente

03. Tipografía

Combinación de fuentes que transmite elegancia en títulos y máxima legibilidad en contenido

Playfair Display - Títulos

Serif moderna y elegante para crear impacto visual en encabezados
Transforma tu vida
Coaching Profesional
Servicios Premium

Inter - Cuerpo de Texto

Sans-serif optimizada para lectura en pantallas, máxima legibilidad

Párrafo grande para introducciones y destacados importantes que capten la atención del visitante.

Párrafo normal para el contenido principal, optimizado para lectura cómoda en cualquier dispositivo.

Texto pequeño para metadatos, fechas y información complementaria.

Implementación:
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;500;700&family=Inter:wght@300;400;500;600&display=swap');

h1, h2, h3 { font-family: 'Playfair Display', serif; }
body, p, div { font-family: 'Inter', sans-serif; }

10. Implementación

Guía técnica para desarrolladores y pasos de migración a frameworks

Comandos de Desarrollo

# Desarrollo con watch
pnpm run dev

# Build para producción
pnpm run build

# Ver documentación
ii .\ui-kit.html

Migración a Laravel

• Crear Blade components para cada elemento del UI Kit
• Usar Vite para compilar assets con Tailwind
• Mantener variables CSS para personalización fácil
• Configurar auto-purge del CSS no utilizado

Migración a WordPress

• Crear bloques Gutenberg con los componentes
• Enqueue del CSS compilado en functions.php
• ACF para campos personalizables en componentes
• Mantener responsive design y performance
